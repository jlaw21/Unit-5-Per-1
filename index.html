Text Formatting

There are three main types of text formatting. Those are: bold, italics, and underline.

Block Elements vs. Inline Elements

A block element moves the content to a new line. The paragraph element is a block element. Each paragraph will start on a new line. An inline element does not move to a new line. It keeps its content on the same line as previous content. The bold, italics and underline elements are all inline elements.

Images

An image element is an inline element which inserts pictures into the document. An image element is self-closing, which means it does not have a closing tag. It has one attribute, src, which is a filepath that points to the image location. Those filepaths can be either absolute or relative.

Simple Image Example

Image Attributes

There are 4 attributes commonly used for images. width and height are used to resize the images. alt allows to use text-descriptions of images for accessibility and broken images. title allows a hover-over effect and is used in place of alt if there isn't one.

Width and Height

The width and height can be measured in many different ways, but more common ones are pixels and percentages. Using pixels, you set the specific, precise size based on pixels. Using percentages, you set the dimensions based on a percentage of the entire page. Percentage re-sizing will often respect aspect ratio while using pixels may not.

Alt Attribute

EVERY picture should at LEAST have the alt attribute. The alt attribute helps with screen readers for visually impaired user and in the cases that the image is broken.

Relative vs. Absolute

An absolute filepath includes all the navigate to get to the server, folders and file. The examples above are all absolute filepaths. An absolute filepath is used when the image is stored on a different server or device than the HTML document.

A relative filepath only include the folder and file. The examples below are all relative filepaths. A relative filepath is used when the images are stored alongside the HTML document.

Links

The anchor element is an inline element and is a way to include hypertext links, otherwise known as just links, in your document. An anchor is a reference point for other documents or files. It commonly has 3 attributes: href, target and download.

Simple Link

LINK

In order for a link to work is MUST include the href attribute. This attribute is the filepath, either absolute or relative, to the document, website or file you wish to link.

Links Attributes

The target attribute allows you to decide where the link opens. There are 4 options, but only one is commonly used. _blank opens the linked document in a new window or tab. _self opens the link in the current window or tab. This is the default behavior. The two others are only used if you have a frame or other window INSIDE a window. We will not be discussing these.

Lists

There are two main types of lists. An unordered list is a bulleted list of items. An ordered list is a numbered list of items. In an ordered list, the style of the list can be altered using attributes list-style-type and type. You can also change the order and numbering of an ordered list by using the attributes start, reversed and value.

An ordered list has multiple attributes broken into two sets. The first set of attributes determines the type of numbering to use. The second set determines values for the items in the set.

The attribute type determines what numbering system to use. There are 4 different types:

  • type = 1, number (default)
  • type = A, uppercase letters
  • type = a, lowercase letters
  • type = I, uppercase Roman numerals
  • type = i, lowercase Roman numerals
