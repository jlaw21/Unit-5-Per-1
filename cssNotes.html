External CSS Document

CSS stands for Cascading Style Sheet. It applies style in the form of colors, fonts, spacing and much more. We have already seen some CSS styling in the inline styles notes we did earlier. However, this is not how CSS is typically done. CSS typically has its own file, a stylesheet, which defines the style for any HTML element you wish.

To begin, we will need to create a new file in this project. The file is commonly called style.css. Once we have created a CSS Stylesheet, we must LINK IT to the HTML documents. This link goes in the head section at the top of the document.

CSS Syntax

CSS is written similar to Java, in that each statement must end in a semi-colon. Brackets are used to denote a group of style for one element. Each block of style must first select a HTML element, called the ,selector. ANY HTML element can be a selector. The style below shows how to select all paragraph elements and set the text color to blue.


        p{
            color:blue;
        }

Hierarchy of CSS

Hierarchy mean, if two different styles attempt to style the same element, hierarchy determines which is shown. In CSS, and in most languages, hierarchy is determined by what gets processed last. The head section of pages gets processed first, then the HTML get loaded and processed. So any inline CSS will override any CSS in an external document.

Review Known Styles

color changes the foreground color. background-color changes the background color. font-family changes the font style and font-size changes the size of the font.

Custom Selectors

Sometimes we want the same styling on two different HTML elements or change the styling of an element which is inside another element. We can select more than one element by separating each with a comma in the selector of the CSS. The code below selects both the paragraph element and the un-ordered list elements.

Selecting Multiple Elements


        p, ul{
            color: blue;
        }

Google Fonts

Browsers are equipped with a select number of fonts. Most of these are very plain and boring. However, Google has created a way to import fonts to use for website. Google Fonts even includes easy documentation on how to include these fonts on your pages.

Styling Links

Each link has 4 different states: unvisited, visited, mouse over(hover), and selected. Each of these states can have their very own CSS. The hover effect can even be applied to MOST OTHER HTML elements. To style a link all 4 of these states must be present or some of them won't work. However, the hover effect can be used by itself on other elements.


    /*unvisited*/
    a:link{
        ...
    }

    /*visited*/
    a:visited{
        ...
    }

    /*hover*/
    a:hover{
        ...
    }

    /*selected link - which is clicking the link but not visiting the site.*/
    a:active{
        ...
    }
